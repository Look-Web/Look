Look

An image sharing website and social network

View project on GitHub

Built with Java

Look is coded in Java 1.8 using JDBC, Apache Commons, and PrettyTime libraries. The front-end is developed using the Foundation front-end framework.

Contributors

Look development is lead by Kevin Holland (@kholland950) with help from Trey McDeane (@tmcdeane), Mark Hensler (@mhensler88), and Matt Fonner (@mfonner).
