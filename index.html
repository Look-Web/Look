What is it?

Look is an image sharing website and social network focused on giving users the ability to easily share and explore images. User's can comment and vote on each other's images and keep in touch by messaging each other.

Built with Java

Look is coded in Java 1.8 using JDBC, Apache Commons, and PrettyTime libraries. The front-end is developed using the Foundation front-end framework.
